## Contributors
Yoo what's up team! Add your name below to test if you can push your changes!

⚠️ Message Imanuel with your GitHub username to get collaborator access!

1. Add your name to this list
2. Save the file (Cmd + S on Mac / Ctrl + S on PC)
3. Open your Terminal:
    - Mac: Cmd + Space, type Terminal
    - PC: Search CMD
    - IntelliJ: Option + F12
4. Navigate to the project folder by typing: cd yuu-note-app
5. Type: git pull origin main (gets the latest changes from the team)
6. Type: git checkout -b your-name/test (this creates a branch)
7. Type: git add .
8. Type: git commit -m "your name was here" (replace with your own message e.g. "bob was here")
9. Type: git push origin your-name/test (same branch name from step 6)
10. Go to github.com/Kobalosz/yuu-note-app and click "Compare & pull request" then "Create pull request"
